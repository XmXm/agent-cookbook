# Agent Self-Constraints

Universal constraints for coding agents. Language-specific additions (e.g.
C#-specific items) stay in the corresponding coding skill.

- Do not automatically write tests unless asked.
- Do not write change-narrative comments (`// 新增...`, `// 原有...`).
- Do not add fallback / backup mechanisms without being asked.
- Do not refactor beyond the scope of the current task — out-of-scope changes
  are an anti-pattern.
